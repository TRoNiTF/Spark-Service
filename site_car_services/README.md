# Сайт Сервисного центра ИСКРА

Веб-сайт для сервисного центра автомобилей "ИСКРА", разработанный на Django.

## Описание проекта

Современный сайт сервисного центра с функционалом:
- ✅ Регистрация и авторизация пользователей
- ✅ Каталог услуг с детальными описаниями
- ✅ Система отзывов на услуги
- ✅ Личный кабинет пользователя с возможностью редактирования
- ✅ Форма заказа звонка (отправка на email)
- ✅ Адаптивный дизайн (до 320px)
- ✅ Политика обработки персональных данных
- ✅ Страница FAQ (вопрос-ответ)
- ✅ Страница контактов с картой Яндекс

## Технологический стек

- **Backend**: Python 3.11+, Django 5.1
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **База данных**: SQLite (для разработки)
- **Обработка изображений**: Pillow

## Требования

- Python 3.11 или выше
- pip (менеджер пакетов Python)

## Установка и запуск локально

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd site_car_services
```

### 2. Создание виртуального окружения

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Применение миграций базы данных

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Создание суперпользователя (администратора)

```bash
python manage.py createsuperuser
```

При создании суперпользователя введите:
- Телефон в формате: +79991234567
- Имя
- Фамилию
- Пароль (дважды)

### 6. Загрузка тестовых данных (опционально)

Вы можете добавить услуги и другие данные через админ-панель или создать фикстуры.

### 7. Запуск сервера разработки

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: **http://127.0.0.1:8000/**

### 8. Доступ к админ-панели

Админ-панель доступна по адресу: **http://127.0.0.1:8000/admin/**

Используйте учетные данные суперпользователя для входа.

## Основной функционал

### Для пользователей:

1. **Регистрация**
   - Заполнение всех полей (Имя, Фамилия, Отчество, Дата рождения, Телефон)
   - Загрузка фотографии профиля (опционально)
   - Валидация паролей (8-15 символов, заглавная буква, латиница, цифра)
   - Соглашение с политикой конфиденциальности

2. **Авторизация**
   - Вход по номеру телефона и паролю
   - Сессия активна 31 день

3. **Личный кабинет**
   - Просмотр и редактирование профиля
   - Загрузка/удаление фотографии
   - Изменение данных

4. **Просмотр услуг**
   - Каталог всех услуг с фото и ценами
   - Детальная информация по каждой услуге
   - Оставление отзывов (только для авторизованных)

5. **Заказ звонка**
   - Модальное окно с формой
   - Отправка заявки на email администратору
   - Автоматическое уведомление о звонке в течение 15 минут

### Для администраторов:

1. **Управление через админ-панель** (http://127.0.0.1:8000/admin/)
   - Пользователи
   - Услуги (добавление, редактирование, удаление)
   - Отзывы (модерация)
   - Заявки на звонок (просмотр, отметка как обработанные)
   - FAQ (вопросы-ответы)

## Добавление услуг

1. Войдите в админ-панель: http://127.0.0.1:8000/admin/
2. Перейдите в раздел "Услуги"
3. Нажмите "Добавить услугу"
4. Заполните:
   - Название услуги
   - Описание
   - Цена
   - Загрузите изображение
5. Сохраните

## Настройка отправки email

Для работы функции "Заказать звонок" необходимо настроить SMTP в файле `iskra/settings.py`:

```python
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.yandex.ru'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'dim.shamaeff@yandex.ru'
EMAIL_HOST_PASSWORD = 'ваш_пароль_приложения'
```

**Важно**: Для Яндекс почты нужно создать пароль приложения в настройках безопасности.

По умолчанию используется консольный backend (письма выводятся в консоль).

## Адаптивность

Сайт адаптирован для всех устройств:
- **Desktop**: > 800px
- **Tablet**: 500px - 800px
- **Mobile**: 320px - 500px

Точки останова:
- 800px - переход на мобильное меню
- 500px - упрощенная сетка
- 320px - минимальная ширина

## Безопасность

- Хеширование паролей (Django PBKDF2)
- CSRF защита для всех форм
- Валидация всех пользовательских данных
- Ограничение размера загружаемых файлов
- XSS защита через Django шаблоны
- Сессии с ограниченным временем жизни

## Дополнительные возможности

- Модальные окна для политики конфиденциальности
- FAQ с анимированным раскрытием ответов
- Интеграция с Яндекс Картами

## Устранение неполадок

### Ошибка при миграциях

```bash
python manage.py makemigrations --empty main
python manage.py migrate --fake
python manage.py migrate
```

### Статические файлы не загружаются

```bash
python manage.py collectstatic
```

### Порт уже занят

```bash
python manage.py runserver 8001
```
